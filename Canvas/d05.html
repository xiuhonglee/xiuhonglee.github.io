# 1.1 方法一

  • 完整代码

  • 先指定剪辑区域

// 指定剪辑区域
this.ctx.beginPath();
this.ctx.arc(100, 75, 50, 0, Math.PI * 2, false);
this.ctx.clip();

// 填充剪辑区域
this.ctx.fillStyle = "#ffffff";
this.ctx.fillRect(0, 0, this.canvas.width, this.canvas.height);

TIP

剪辑区域会保留在上下文变量中，使用时建议使用context.save()及context.restore()包裹起来，防止污染全局上下文。
